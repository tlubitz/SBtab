    Conventions for structured data tables in Systems Biology

    SBtab is a set of conventions for data tables, facilitating data exchange in Systems Biology. Its aim is to simplify data processing by a standardised syntax and names. This includes, among other things, prescriptions for column names, semantic annotations, and physical units.

    On this website, we present SBtab in detail and offer a range of online tools (such as the SBtab validator, an SBML converter, and the SBtab customiser), next to a variety of example files. If you have not worked with SBtab before, we recommend our section Getting Started.
